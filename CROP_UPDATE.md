# 📝 마우스 드래그 크롭 기능 업데이트

## 🎉 새로운 기능: 마우스 드래그 크롭

마우스로 직관적으로 크롭 영역을 선택할 수 있는 시각적 크롭 기능이 추가되었습니다!

### ✨ 주요 특징

#### 🖱️ **마우스 드래그 크롭**
- 이미지 위에서 마우스로 드래그하여 크롭 영역 선택
- 실시간 시각적 피드백 (반투명 주황색 영역 표시)
- 드래그한 영역의 좌표와 크기 자동 계산

#### 📐 **수동 입력 크롭**
- 정확한 픽셀 좌표 직접 입력
- 빠른 설정 버튼 (정사각형, 16:9, 4:3 비율)
- 수치 기반 정밀 제어

#### 🔧 **미세 조정 기능**
- 마우스 드래그 후 숫자로 좌표 미세 조정 가능
- 드래그와 수동 입력의 장점 결합

### 🎯 사용법

1. **✂️ 이미지 자르기** 섹션 펼치기
2. **크롭 방식 선택**:
   - 📐 수동 입력: 숫자로 정확한 좌표 설정
   - 🖱️ 마우스 드래그: 시각적으로 영역 선택
3. **마우스 드래그 방식**:
   - 이미지 위에서 마우스로 사각형 그리기
   - 여러 번 그릴 수 있으며 마지막 사각형이 선택됨
   - 🗑️ 버튼으로 모든 그림 지우기
4. **실시간 미리보기**: 선택한 영역이 즉시 미리보기로 표시
5. **🔽 잘린 이미지 다운로드**: PNG 파일로 즉시 다운로드

### 🛠️ 기술적 개선사항

- `streamlit-drawable-canvas` 라이브러리 활용
- 고해상도 이미지 자동 스케일링 (최대 600px 폭)
- 좌표 변환 및 경계 검증
- 실시간 크롭 미리보기
- 향상된 UI/UX 디자인
